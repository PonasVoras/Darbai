# Word or Paragraph hyphenation PHP CLI



``` sh
start by : php HyphenationApp.php
# there are going to be database options provided
Options : 
	-y, for using the database
	-n, for not using the database
# there are going to be cache options provided
Options : 
	-y, for cleaning the cache
	-n, for not cleaning the cache
# there are going to be hyphenation options provided
Options : 
	-w, for word
	-p, for paragraph
For word hyphenation word input will be required
For paragraph hyphenation sentence has to be saved to data/paragraph.php
```
